# Custom Boilerplate Library: ```Express``` Branch


## To Start the Node (w/ Babel) App

```bash
npm start
```

---

## To Test Routes

```bash
# GET ROUTE
curl http://localhost:3000

# POST ROUTE
curl -X POST http://localhost:3000

# PUT ROUTE
curl -X PUT http://localhost:3000

# DELETE ROUTE
curl -X DELETE http://localhost:3000
```
